# 🚀 Automatic Attendance System using Face Recognition with 2FA

## 📌 Project Summary
Developed a **secure, AI-powered attendance management system** using **Face Recognition and Multi-Factor Authentication (2FA)** to eliminate proxy attendance and improve accuracy in real-time environments.

This system integrates **Computer Vision, Machine Learning, and Web Development** to provide a scalable and reliable attendance solution for educational institutions and organizations.

---

## 💡 Key Highlights

- ✅ Built using **Python, Flask, OpenCV, Scikit-learn**
- 🔐 Implemented **Multi-Factor Authentication (Face + Voice + OTP)**
- 📊 Achieved **98.7% authentication accuracy**
- ⚡ Reduced manual attendance effort by **92%**
- 🚫 Eliminated proxy attendance using **liveness detection**
- 📈 Real-time analytics and reporting system

---

## 🧠 Problem Statement

Traditional attendance systems (manual, RFID, biometric) are:
- Error-prone  
- Time-consuming  
- Vulnerable to proxy attendance  

This project solves these issues by introducing a **contactless, automated, and secure attendance system**.

---

## 🏗️ System Design

### 🔄 Authentication Flow
1. Face Recognition (Primary)  
2. Voice Authentication (Secondary)  
3. OTP Verification (Fallback)  

Ensures **high accuracy and reliability under all conditions**.
